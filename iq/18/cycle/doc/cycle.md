NAME
====

**cycle** - detect cycle in a singly linked list

DESCRIPTION
===========

Problem
-------

Given a singly linked list, report whether it has cycles. Do not use a hash set to keep track of visited nodes.

Example
-------

Input:

```
1 - 2 - 3 - 4
    ^       |
    |       |
    +-------+
```

Has cycle from item 4 to item 2
