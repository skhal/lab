NAME
====

**midpoint solution** - find midpoint in a singly linked list

DESCRIPTION
===========

Use two pointer to scan the list at different speeds: one pointer moves two items, while the second one make a single item step.

Complexity:

-	*Time*: O(n)
-	*Space*: O(1)
