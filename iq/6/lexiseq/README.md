NAME
====

**lexiseq** - next lexicographical sequence

PROBLEM
=======

Given a string of lowercase English letters, form a new string with rearranged letters to represent the next lexicographical (alphabetical) order. If the string is the last in the lexicographical order, return the first one.

EXAMPLE
=======

One
---

*Input*: "abcd"

*Output*: "abdc"

Two
---

*Input*: "dcba"

*Output*: "abcd"
