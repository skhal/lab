# NAME

**shiftzeros** - shift zeros to the end of a collection


## PROBLEM

Given an array of numbers, perform in place move of zeros to the end of the
array. Maintain the relative order of non-zero elements.

## EXAMPLE

*Input*: `[0, 1, 0, 3, 2]`

*Output*: `[1, 3, 2, 0, 0]`
