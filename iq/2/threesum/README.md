NAME
====

**threesum** - sum of triplets

PROBLEM
=======

Given an array of numbers, find unique triplets that sum up to zero. Unique means that any permutation of numbers in the triplet is considered the same triplet, e.g. [1, 2, 3] and [2, 1, 3] are the same triplets.

EXAMPLE
=======

-	Input: [-2, -3, 5]
-	Output: [-2, -3, 5]
