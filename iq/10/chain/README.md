NAME
====

**10** - find longest chain in a collection

DESCRIPTION
===========

PROBLEM
-------

Given an array of numbers, find the longest chain of consecutive numbers. Two numbers are consecutive if they have a difference of 1.

EXAMPLE
-------

**Input**: [7, 1, 8, 9, 2, 12]

**Output**: [7, 8, 9]
