NAME
====

**14** - intersection of two singly linked lists

DESCRIPTION
===========

Problem
-------

Find an intersection of two singly linked list, i.e., a common node, shared by both lists:

Example
-------

Input:

```
[1] -> [2] -+
            |
            v
           [3] -> [4]
            ^
            |
       [5] -+
```

Output: Node with value 3
