# NAME

**nuc.lab.net** -- Intel NUC server with FreeBSD OS


# DESCRIPTION

The hardware is Intel NUC Kit NUC6i7KYK with:

  * RAM: 2x 16 GB DDR4-2400 16GB  
    HyperX HX424S14IBK2/32 kit

  * SSD: 2x 1TB NVMe  
    WD WDS100T3X0C Black SN 750 Gen3 PCIe M.2 2280
