NAME
====

**home** - user home configuration files.

DESCRIPTION
===========

`home` holds user configurations files like `~/.vimrc`, `~/.tcshrc`, etc.

Install
-------

```console
% make -C home
.. install tcshrc to /home/skhalatyan/.tcshrc
.. install vimrc to /home/skhalatyan/.vimrc
```
