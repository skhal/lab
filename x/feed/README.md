Name
====

**feed** - streaming feed reader

Description
===========

The project implements a streaming feed reader, inspired by Sameer Ajmani's Google I/O 2013 [talk](https://go.dev/blog/io2013-talk-concurrency) "Advanced Go Concurrency Patterns".

See [cmd/feedreader](./cmd/feedreader/) for use
