# NAME

**kscope** - compiler frontend

# DESCRIPTION

Kscope implements a compiler frontend for a toy language *kaleidoscope*.
The project follows
[LLVM tutorial](https://llvm.org/docs/tutorial/MyFirstLanguageFrontend/index.html)
on the subject.
